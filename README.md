<!-- ABOUT THE PROJECT -->
<h2 id="about-the-project"> :pencil: About The Project</h2>

<p align="justify"> 
 The objective of this project is to extract Airbnb data from <a href="http://insideairbnb.com/get-the-data/">Get the data</a> in the form of csv's and store it in a file using automated code. The reviews provided by the Airbnb guests are then processed using an identifier to classify them as positive or negative. To save time on computational efforts, the csv is updated with an additional field that contains the outcome of the identifier for each review.
</p>
